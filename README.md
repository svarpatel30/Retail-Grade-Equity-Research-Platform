A transparent, retail-focused equity research tool that allows users
to analyze public companies using real financial data and a
clear, assumption-driven valuation framework.

## Features (MVP)
- Company overview
- Financial statements
- Key performance metrics
- Discounted Cash Flow (DCF) valuation
- Bull/Base/Bear scenario analysis
- Methodology and limitations disclosure

## Philosophy
This platform prioritizes clarity, transparency, and financial reasoning
over black-box predictions or automated recommendations.

## Data Sources
- Public company filings
- Yahoo Finance (for raw data aggregation)

## Disclaimer
This tool is for educational purposes only and does not constitute
investment advice.
